Interleaving

• Idea: rather than showing each user the results from just one of the 2 systems, the results are combined in an unbiased way so that every user can observe results produced by both systems;

• If one system returns better results on average, users can select them more often;

• Interleaved comparison methods consists of two components:

  • Mixing policy: method to select documents from the original rankings and construct the interleaved lists;
    • Scoring rule: method for inferring comparison outcomes, i.e., how to interpret observed interactions;

• Based on hypothesis that the constructed interleaved result should minimally differ from either candidate ranking

Convolution in the spatial domain is equal to multiplication in the frequency domain, and
